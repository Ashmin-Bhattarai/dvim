usage
terminal 
# open nvim in current directory
$ dvim

# open a specific file
$ dvim myfile.py

# install a specific image tag
$ curl -fsSL https://get.dvim.ashmin.info.np/install.sh | bash -s v1.0

# pull latest image and update launcher
$ dvim update
configuration

dvim uses two optional Lua config files. Drop them on your host — no image rebuild needed. Project config takes priority over system-wide config.

global ~/.config/dvim/user.lua 
-- options
vim.opt.relativenumber = true
vim.opt.scrolloff = 8

-- keymaps
vim.keymap.set("n",
  "<leader>q",
  "<cmd>q<cr>")
project .dvim.lua 
-- project-specific overrides
vim.opt.tabstop = 2
vim.opt.shiftwidth = 2

-- project-local keymap
vim.keymap.set("n",
  "<leader>r",
  "<cmd>!make<cr>")
